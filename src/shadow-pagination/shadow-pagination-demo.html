<shadow-pagination>

About

<shadow-pagination> is a web component created by custom elements, shadow DOM and HTML template APIs. It's non-dependent.

The prefix shadow- means nothing, it is because a valid custom element name must contains the "-".

Install

Usage

Default usage


      <shadow-pagination></shadow-pagination>

What happened? Sure, the pagination must require a total page number, here, it is count


      <shadow-pagination count="12"></shadow-pagination>

bingo!
